#Digital services-Usage-Analysis
This project analyzes Digital services usage using Python. It involves data cleaning, reshaping wide-format data, identifying top-used platforms, visualizing usage patterns, and comparing behavior across user clusters to demonstrate practical data analytics skills.

## Key Steps
- Data loading and understanding
- Data cleaning (duplicates, irrelevant columns)
- Reshaping wide-format data to long format
- Identifying top-used Digital service
- Basic visualization and cluster-level insights

## Tools
- Python
- Pandas
- Matplotlib
- Google Colab
